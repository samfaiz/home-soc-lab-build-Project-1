================================================================================
 BACKUP / RESTORE
================================================================================
 
 A full XML backup of this configuration is committed alongside this file as:
   pfsense-config.xml
 
 To restore:
   1. Diagnostics -> Backup & Restore -> Restore configuration
   2. Browse to pfsense-config.xml
   3. Click "Restore configuration"
   4. pfSense reboots automatically
 
 NOTE: the XML backup has been sanitized — the admin password hash has been
 replaced with a placeholder. After restoring, set a new password via
 the console (option 3 — Reset webConfigurator password).
